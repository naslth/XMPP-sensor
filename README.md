## Project gồm có:

### Openfire

### MySql

### Backend:

- Java (Spring Boot)
- Maven

### FrontEnd:

- React.Js

## Khởi chạy dự án

- Clone project
- Start Openfire trên máy
  - Config openfire
- Cd vào thư mục backend/src/main/java/ . Chạy file BackendApplication
- Cd vào thư mục client-chartjs.
  - `npm install `
  - `npm start`
